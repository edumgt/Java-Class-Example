# 학습 커리큘럼 재정의 (Java-Class-Example-020)

이 문서는 현재 저장소의 실습 코드를 **학습 난이도 기준으로 단계화**한 로드맵입니다.

## 난이도 레벨

- **Lv.0 입문**: 문법 감 잡기, 실행 환경 적응
- **Lv.1 기초**: 조건문/반복문/메서드/배열
- **Lv.2 중급 기초**: 클래스/객체/컬렉션/예외/입출력
- **Lv.3 실무 기초**: 파일 구조화, 인덱스, 데이터 모델 이해
- **Lv.4 도메인 응용**: 시뮬레이션, 거리 계산, 비용 로직
- **Lv.5 확장 실습**: DB 연동, 문서 자동화, 패키징

## 권장 학습 순서

1. **기초 문법/객체지향 학습 (Lv.0~Lv.2)**
   - 목표: 문법/객체지향/기본 API를 빠르게 체득
   - 산출물: 작은 단위 실습 소스 직접 작성

2. **ISAM 저장/검색 학습 (Lv.3)**
   - `com.sample.ISAMExample`
   - `com.sample.ISAMExample2`
   - 목표: 고정 길이 레코드, 파일 오프셋 인덱스 감각 습득

3. **선박 시뮬레이션 학습 (Lv.4)**
   - `com.sample.ShipSimulationISAM`
   - `com.sample.ShipSimulationUnload`
   - 목표: 좌표/거리 계산 + 상태 기반 비즈니스 로직 구현

4. **외부 시스템 연계 학습 (Lv.5)**
   - `com.ship.UpdateShipLocation` (MongoDB)
   - `com.sample.CreateHwpExample` (HWP 생성)
   - 목표: 저장소/문서 라이브러리 연동 실전 감각 확보

## 학습 운영 방식

- 각 단계마다 `README`의 실행 명령으로 직접 실행
- 단계 종료 시 체크리스트
  - 코드 설명을 말로 할 수 있는가?
  - 입력/출력 구조를 다이어그램으로 표현 가능한가?
  - 핵심 로직을 변형(파라미터 변경)해도 동작을 예측 가능한가?
